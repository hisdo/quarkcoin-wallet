En utilisant cette boîte de dialogue, vous pouvez recevoir des Dimecoins (vous devez avoir installé Dimecoin Wallet).

D'abord, saisissez le montant de Dimecoins à recevoir.

Ensuite, l'autre personne doit scanner le code QR généré. Ou bien, vous pouvez envoyer une URL Dimecoin à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL Dimecoin.

Dans les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des Dimecoins avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer le paiement.
